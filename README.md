# Event Kvartal

Статическая версия нового сайта для event-группы «Спутник / Event-квартал».

## Быстрый старт

```bash
cd ~/code/event-kvartal
python3 -m http.server 8010
```

Открыть:
- `http://localhost:8010` — сайт
